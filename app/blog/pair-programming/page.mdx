# 结对编程

仍能记起两年前和睿君共用一台电脑编程时奇高的效率，仍能记起两年前和浩哥哥在不断讨论不断迭代的精彩瞬间，仍能记起一年前和杰兄、启盛合作时的畅快淋漓。

在我们将那版冠名为为极限与终极的时候，在我们故作轻松地说出已经赢了的时候，在一个人犯错另一个人立刻指出的时候，在一个人的工作记忆趋于极限而另一个人不经意指出主要矛盾的时候......我都非常好奇，是什么能如此有效地凝聚着注意力，切实地提高生产力？

直到今天，我才恍然大悟，那叫结对编程,是敏捷开发的最佳实践之一，输入代码的人称作驾驶员，审查代码的人称作观察员。像极了赛车手与领航员的关系，在副驾驶有条不紊地读着路书时，赛车手正操纵那头狂暴的野兽，步步为营地融入心流。
