# 🦹网络安全赏金猎人

> *`网络安全赏金猎人`在资本主义国家更为常见，往往是一群无政府主义者，黑帽、白帽、灰帽都有，凭借黑客技术为雇主取得敏感信息或者完成勒索行动以获得赎金，他们是一群密码朋克，掌握着高超的渗透技术和社会工程手段，自由地穿梭在网络世界中。*

我迷迷糊糊地醒来，头有点小痛，嘴巴则干的要命。

抓过眼镜戴上，从懒人沙发上坐起来，桌子上留着吃了一半的泡面，面前的显示器上开着十几个终端，留着没写完的代码。

把剩下的鸡尾酒喝光了以后，意识稍微清醒了一点，去楼下的罗森买了早餐，看了看时间，现在是凌晨五点钟。

我拐进臭巷子，妓女们都下班了，昨晚喝吐的大哥瘫坐在贴满小广告的电线杆旁边，呕吐物覆满了裤腿，我继续往巷子尽头的盘古网吧走，不过目的地倒不是那里，我穿过旁边的小门，进到一个地下室里面。

![](https://img.3dmgame.com/uploads/images/news/20210516/1621153649_466183.jpg)

人到的挺齐，大家都还在猛敲键盘，见到我来了，文森特马上给我腾了一个工位出来，给我找了把机械键盘和一个显示器。我启动了电脑，和他寒暄了几句。

从内网的 git 仓库里面拉取了最新的提交记录，我开始着重分析了几个新的日志文件，我负责根据已有的服务器指纹和社工库写出 exp 脚本的大致框架，文森特则负责帮我完善脚本的细节与异常处理问题，我们配合的很不错。

我现在身处的地下室里大概有十几个哥们，是一个网安 DAO，几乎都是灵活就业人员，技术水平将就，平时我们组队打一打有奖金的 CTF，挖挖 SRC，参加一下护网，也就是混口饭吃的样子。

上个星期接到一单，让我们去渗透一家互联网公司，雇主应该是他们的竞争对手，需要一个项目的核心代码。

我们正在为明天的近源渗透做准备，十几个人分成三个小组，我的临场反应能力极其差劲，一紧张敲键盘就容易出错，所以负责后勤，现在和文森特一起，正在写一个内网跳板软件，拷贝到 BadUSB 中，以便插入以后可以穿透防火墙。

至于那群负责硬上的哥们，鬼知道他们有什么手段可以混进去，据说之前有一次，负责社会工程的强哥被保安发现了，就拿出了他以前偷到的渗透测试安全许可，一顿瞎就把忽悠把保安骗过去了，我想要是我在那个场景下，肯定紧张得要命，现在说不定已经在牢里坐着了。

有同行讲，黑产的东西别碰，不过我总是嗤之以鼻，在这个狗屁年代，谁不是见钱眼开，那群红队的待在公司里当成猪养着，要是哪天出事了背完锅，看他还能不能再憋点漂亮话出来！

不多说了，我还要继续工作了，这次的奖金是一笔大钱！

# 🧳数字游民

> *`数字游民`是一群追求自由、灵活和自主生活方式的个体。他们不受传统工作地点的束缚，通常从事创造性和趣味性的工作，能够在全世界自由流动，以实现工作和休闲的动态平衡。*

我叫阿江，是一个数字游民，你可以用网络上的ID称呼我————`Friday`。
        
我现在在尼泊尔做一名背包客，目前是晚上十点，我正在一座深山里和我前天认识的来自英国的朋友Johny一起坐在篝火旁聊天。

我是一个自由软件开发者，从事的是Web3方向的开发，也做全栈，上个月去台湾参加了一个Telos生态的黑客松，由于我大学期间学的是电子信息工程，所以对物联网和嵌入式的开发也比较拿手，所以我们团队这一次做的是RWA方向的，为边缘AI创建一个通过代币来分配算力的平台，并且优化已有的2FA机制，我觉得这是一件很有挑战性的工作，最后我们团队拿下了三等奖，每个人获得了大概一千块的奖金。

然后我便踏上了这次穷游，一场说走就走的旅行！。我之前去过不少城市，清迈、巴塞罗那、伦敦、哥本哈根、布达佩斯......当然，我也不是一直都在玩，我一边旅行一边在社区里物色工作，或是为开源项目贡献一些代码，收到一点Sponsers的赞助，当一些零花钱咯，偶尔也到当地去打打零工，生活节奏总是很慢。

我同时活跃在国内的各大数字游民社区，与其他同好们保持着密切的联系，Johny就是我前天在`NomadList`网站上认识的。

他说想来山里做一次背包客，于是咱租了一辆吉普车开到沿着山麓一直走，穿过了茂密的森林，树木高大而挺拔，阳光透过树叶的缝隙，洒在蜿蜒的小路上，形成了斑驳的光影，经过清澈的溪流，水流潺潺，偶尔还能看到一些野生动物在河边饮水。这里的自然风光让人心旷神怡。

我们聊到了工作，Johny是一个UX设计师，现在还在GAP当中，他说他正准备创业，我一边吃着烤肉一边听他讲完了他疯狂般的想法，俩人一拍即合，随即决定回去以后大干一场。

有了！脑海里，一个疯狂的想法诞生了，我掏出手机开始打字，脑海里的设计图越发越清晰。

# 🧑‍🔬心理学家

> `认知心理学`，20世纪50年代中期在西方兴起的一种心理学思潮和研究方向。广义指研究人类的高级心理过程，主要是认识过程，如注意、知觉、表象、记忆、创造性、问题解决、言语和思维等。狭义相当于当代的信息加工心理学。即采用信息加工观点研究认知过程。
> 
> `社会心理学`,研究个体和群体在社会相互作用中的心理和行为发生及变化规律。社会心理学在个体水平和社会群体水平上对人际关系进行探讨。在个体水平上进行研究的内容有：个体社会化过程，交往，言语发展，伙伴，家庭和居住环境及学校对个人的影响等。

张教授品了一口奶咖，语重心长地望着我，“那件事，你考虑好了吗？”

我轻轻搅动着杯中的拿铁，深吸了一口气，“教授，我对认知心理学的热爱从未减退，尤其是智商G因素的研究，它让我着迷。但您也知道，我对人类如何在社会环境中构建自我认知，以及这些认知如何影响他们的行为和决策，同样充满了浓厚的兴趣。”

“我明白你的兴趣所在。社会心理学的研究确实能够揭示人类行为的深层动机。但认知心理学，尤其是智商G因素的研究，对于理解个体差异和教育策略同样至关重要。”

我皱了皱眉头，缓缓道：“我一直在思考如何将这两种兴趣结合起来。也许，通过研究个体在社会互动中的认知过程，我可以找到两者之间的联系。”

张教授的手机响了，他看了看来电显示，笑着对我说：“是关于那个超级中学的合作项目。”他接起电话，简短地交谈了几句后，便挂断了。

他转过头来，语气中带着一丝不易察觉的期待：“刚才的电话是关于我们即将开始的项目。他们希望我们能够评估一种新的教学方法对学生认知能力的影响。我认为这是一个绝佳机会。你有工科背景，接触过数据处理，分析工作交给你，我很放心。”

我点了点头。

张教授沉默了一会儿，“那件事，我尊重你的选择，也支持你追求自己的兴趣。但如果你愿意，可以先参与这个项目，然后再决定你的研究方向。”

“好的，教授，我愿意先参与这个项目。” 我抿了一口咖啡，感觉眼里泛着光。

# 🕵️‍咨询侦探

> 传奇侦探`夏洛克·福尔摩斯`，自称是“咨询侦探”（Consulting Detective），这个职业在现实世界中并不常见，在故事中，它指的是一种私人侦探，通常被其他侦探、警察或个人聘请来解决复杂的案件。

我推开诊所的玻璃门,步入闷热的香港街头。空气中弥漫着一股潮湿的味道,预示着又一场暴雨即将来临。我在附近的长椅上坐下,闭上眼睛,开始梳理刚才从那位年轻医生那里获得的信息。

这起命案已经困扰我整整一周了。死者是一位颇有名望的心胸外科专家,死于一种罕见的神经毒素。初步尸检报告显示,毒素是通过静脉注射进入体内的。这让我不禁想起了几年前那起"注射器杀手"案。当时我作为辩护律师,成功为一名被误认为凶手的护士洗脱了嫌疑。那个案子最终以真凶自杀告终,留下了许多未解之谜。

我睁开眼,掏出笔记本快速记录下几个关键词:硫代胆碱酯酶、丙氨酸转氨酶、γ-谷氨酰转肽酶。这些都是死者血液检查中异常升高的指标,暗示着肝脏和胆道系统受到了严重损害。但奇怪的是,死者生前并没有表现出任何中毒症状。

我的思绪不由得飘向了上个月在法医学研讨会上听到的一个案例。一位患者因误服含有灰黄霉素的药物而引发了急性肝衰竭。灰黄霉素,这个词在我脑海中盘旋。它是一种强效的抗生素,但同时也是潜在的肝毒性药物。如果将其与某些神经毒素混合,会不会产生一种延迟发作的致命毒剂?

我站起身,决定去拜访一位老朋友。他是香港大学医学院的毒理学教授,或许能为我提供一些专业见解。走向地铁站的路上,我回想起今天询问的几位医生。他们的回答都很相似,没有人承认与死者有过节。但有一个细节引起了我的注意:所有人都提到死者最近在研究一种新型的胸腔镜手术技术。

作为一名侦探，同时也是一名律师,我深知证据的重要性。没有确凿的物证,再精妙的推理也无法在法庭上站住脚。我需要找到那把"手术刀",无论它是实体的还是比喻意义上的。也许,答案就藏在死者的研究笔记里。

地铁呼啸而来,我踏上车厢,心中暗自盘算着下一步的调查计划。这个案子远比表面看起来要复杂得多,就像一个精心设计的迷宫。但我相信,只要耐心细致地追查每一条线索,终会找到真相的出口。

毕竟,正如我常对当事人说的那样:在法律和侦查的世界里,devil is in the details。每一个微小的细节都可能是破案的关键。而我,就是那个要把这些细节拼凑起来,还原真相全貌的人。

# 🏃‍➡️连续创业者

> 连续创业者是指成功或失败地操作过不止一个、两个、三个，甚至更多的创业项目或者商业模式的人。

“作为一个从大学宿舍就开始coding的极客,到现在已经连续创立了三家科技公司,我深刻体会到设想和执行之间的巨大鸿沟。我们上一轮A轮融资刚刚收官,估值达到了2000万美元。这无疑证明了市场对我们具身智能技术的认可,但同时也给我们带来了巨大的压力和期待。

“边缘计算: 目前,我们的大部分高级计算都依赖于云端。但是,随着边缘计算技术的进步,我们有机会将更多的AI功能直接集成到车载系统中。这不仅可以减少延迟,还能在没有网络连接的情况下提供更强大的功能...”

作为收尾，我特地停顿了一下，终于做出了那个在脑海里幻想过很多次的帅气动作————微微一笑，伴随着幻灯片的平滑效果将手指朝向了荧幕，然后自信地说道，“这些挑战看似艰巨,但正是这样的挑战让我们的工作如此令人兴奋。记住,当我们第一次实现动态姿态控制算法时,很多人也认为那是不可能的。但通过将卡尔曼滤波器与我们自主开发的机器学习模型相结合,我们做到了。”

“我相信,通过整合最新的SLAM技术、优化我们的神经网络架构,我们不仅能解决这些问题,还能再次重新定义个人交通工具的概念。”

掌声响了起来，我将脸朝向聚光灯，保持那个帅气的pose。

说不定有摄像头正对着我呢，我想。
