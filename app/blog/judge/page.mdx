# 机械时代的宣判

阿泽加完班回到家已经很累了，打了一天螺丝，他只想好好洗一个热水澡，打几把王者，刷一会儿抖音，然后上床睡觉。

可是门口两个站着大盖帽，他们一动也不动。
“您好，请问是陈先生吗？”其中一名警察转向了阿泽，保持着有些渗人的职业微笑。阿泽懵地点了点头。

“您不用紧张，我们想先检查一下您手机上的抖音极速版。”

说罢，另一个警察非常熟练的出示了警察证，说话者接过了阿德解锁以后的手机点开了软件的版本号页，直至看清楚两个小数点以后的数字，他的神情才正式严肃起来。他向另外一个同伴点了点头，然后用冷酷的眼神望向阿泽。
“陈先生，你有看过这次升级的更新日志与用户协议吗？”

这几乎是一个不需要回答的问题。

另一个警察从包里掏出了一个档案袋，然后从档案袋里面掏出了一份体检报告，一份结婚证复印件，一份录用通知书，还有厚厚的一塌合同似的东西。随后他机械地开始念到一段很熟练的台词。

“我们检查到您在2023年4月13日下午十点三十四分在编号为XqSiZxRTEk的抖音视频下发表了言论使用了断言语句判定这对情侣走不到最后，这是他们于今年5月4日在重庆市巴南区民政局办理的结婚证件，这一笔赔偿金约合人民币陆百陆拾陆元柒角；随后您在2024年9月3日上午六点十四分在编号为s3SiZeztyk的视频下方嘲讽了针对这位职高学生辍学创业的经历，这是他今年找到的工作录用通知书，比您的工资要搞出两百个百分点，这意味着您约合赔偿……”

没有听完剩下的宣判，阿泽只感到眼前一黑地昏倒在地。
