# 我从《三体》读到了什么

《三体》这本书给我的启发，着重在四个概念上。这四个概念分别是黑暗森林，执剑人，面壁者和破壁人。

黑暗森林法则是罗辑在叶文洁的提点下推导出的宇宙公理；而执剑人，则是基于这一法则设计出的威慑系统的掌控者。

黑森法则在宇宙维度成立的前提是，宇宙的物质总量保持不变，而文明却疯狂扩张。认识到这一点很有意思，因为“人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾”，翻译成中文，就是社会黑森法则在中国内卷社会成立的前提。

我更进一步的猜想是，人类前额叶皮质的缺陷，使得黑暗森林威慑成为了一切基本人际关系的基础，无论是在通俗社会还是大学校园。

K12教育为在象牙塔中我们这些尚未出身社会的学生群体，提供了太多具有欺骗性的共同体感觉，我相信大家这么多年来，应该都还没有遇见过任何绝对意义的敌人。 

然而，黑暗森林中有限资源和无限欲望的博弈游戏，一定会逐渐渗透进我们的社会生活中——只有当浪潮退去，才知道谁在裸泳了。

由此可见，立足于社会，必然需要威慑手段。

现如今，世俗奉为圭臬的两种经典价值观，是拜金主义和英才至上。对应一下就是钱和才能。前者可被称之为收入；后者亦可称之为文凭。这两种东西，是通俗社会这一黑暗森林中常用的威慑手段，是与其他猎人们直接对决的资本。

除此以外，能够为大多数人，尤其是年轻人，提供情绪价值的内容，也已经被赋予了正义性。擅长在互联网上创作出这种内容的人，正是——网红、明星。所以，他们同样是执剑人。基于Web 2.0的流量经济，使观众可以为他们的热点买单的同时，也在加速变现他们财富这一基本威慑手段；而擅长在现实中创造这种情绪价值的人，他们有高超的情商技巧，在社交圈中具有广泛影响力，同样是执剑人。

情绪价值可以来自校园恋爱，所以帅哥美女是执剑人；情绪价值可以来自你的王者五杀，所以传奇王者是执剑人；情绪价值可以来自一张成绩单，所以学霸和上岸人也是执剑人，有句话是这么说的——“上岸第一剑，先斩意中人”。

巨大的力量同样也伴随着巨大的威胁。执剑人手中的剑，也是一把悬在头顶的达摩克利斯之剑。所以，无数个作风不端正的明星进去了；所以，无数个高启强被搞垮了；所以，百度的AIGC文心一言让我们如此失望；所以，面对孔乙己的长衫，即使强如余华也说不出个所以然。

读大学的意义是什么，黑暗森林可以解释。大学是小社会与大社会的完美过渡，能让一部分人提前窥见黑暗森林的本质，从而提前适应真正的大社会；通过完成学业，稳步发展才能，取得学历这一把剑，进入大社会以后，再通过学历之剑进一步掠取财富之剑，以此确立在社会这一黑暗森林的一席之地。

接下来我将分享的第三个和第四个概念是面壁者和破壁人，面壁者是由PDC指派的四位人类智士，他们要在欺骗智子的同时完成拯救地球的任务。

破壁人，则是ETO的三位负责识破面壁者谋略的对手。唯有罗辑，没有人知道他的威胁在哪里，他需要自己发掘自己的意义。

罗辑成功之前已经耗尽了所有人类的耐心，以至于落得“人类不感谢罗辑”的下场。可是，当他在墓前直接与三体世界对决的时候，形势却瞬间转变了。这种绝地反杀的迅猛之势让人难免好奇他成功的原因。

诚然，他的所作所为绝非好学生按部就班式的答题，甚至纵观整个三体宇宙，另外三位面壁者——雷迪亚兹，希恩斯，泰勒，又或是云天明，韦德，章北海这一类狠角色，他们做的事情，在当时都绝无被任何人理解的可能性。

互联网浪潮之巅的企业家同样如此。

Web 1.0出现之前，所有人都在说，在线广告、流量变现是泡沫空梦，于是有人顶着怀疑去做了，这波人包括马云，马化腾。

Web 2.0出现之前，所有人都在说，创作者激励，知识付费是绝无可能的，于是有人冒着风险去做了，这波人包括张一鸣，陈睿。

ChatGPT出现之前，谁也料不到，AIGC能达到现在这种高度；谁也想不到，AI将彻底重塑世界产业格局。于是OpenAI去做了，做得盆满钵满，做得名利双收。

原子弹最大的秘密，并不在于原子弹怎么被制造出来，而是原子弹能够被制造出来这个既定的事实。

所有面壁者最大的秘密，不是他们知道他们应该怎么做，而是他们知道这么做，一定能成功，他们心中住着“虽千万人吾往矣”的信念。

这种信念，并非是空穴来风，它根植于一种石破天惊的胆识，一种纵横未来的长远的目光，一种泰山崩于前而面不改色的深刻的格局。

这些又是怎么来的？

这不过是在倒逼我们去搞清楚最开始的问题——我们究竟是谁，我们到底想要什么，我们应该怎么做。

而面壁者在无数次面壁，无数次碰壁的过程中，已经给出了他们自己的答案，那么，我们的答案又是什么呢？

这就需要我们像罗辑一样，亲自去做，我们自己的破壁人了。
