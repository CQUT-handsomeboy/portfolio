# 《野心》魔改歌词自评

魔改的歌词，实际上勾勒出了一副弱肉强食，适者生存的残酷校园爱情图景。

黑暗森林是指三体的黑暗森林法则，主人公误入校园爱情这片黑暗森林，化身秉剑的猎人于森林中潜行，他必须小心翼翼，因为森林中很有可能会出现与他相似的猎人——即情敌——觊觎着同一个猎物，即追求的对象。当他发现了对方的存在，唯一能做的就是拔剑消灭之，即一旦发现情敌的存在，就需要与之进行斗争。

在原著中被描绘的场景里，猎人使用的是枪，而此处为什么却被我写成了剑呢，因为从武器的角度出发，枪的杀伤力大，效果迅速显著，然而在猎人之间抗衡也就是情敌的斗争，必定是一场持久战、体力战、拉锯战，更考验博弈的技巧。剑相较于枪而言，更不容易一招置人于死地，而是能逐渐瓦解对手，不仅仅是实体上，更是精神上。

从歌词的人称可以推断，呼告的对象“你”是主人公对猎物的称呼，他与猎物都为世俗保留过初心，这是指主人公此前未驯服过猎物，猎物也没有被其他猎人所驯服。即双方互为初恋。这在某种程度上致敬了《小王子》中狐狸与小王子的恋情。

在这片森林里，规则是俘获猎物的芳心，而非是去杀死它们。因此，这考验猎人的水平。征服的过程，必将是一副残酷血腥的图景，很有可能会被猎物的锋利爪牙所伤。所以说，主人公企图感化猎物的纯粹的深情是愚蠢的。猎物的意图难以揣测，这也是我写到猎物的内心像黑盒一样的原因。黑盒是软件测试的一种方法，指在不清楚内部程式具体逻辑的情况下测试软件。对于每一个猎人而言，他们要做的不仅仅是测试，更是要破解这一模型，即征服他们中意的猎物，这就要求猎人具备一种黑客的思维。显然，主人公不具备这一思维，看不透系统参数之外的制胜之法，因此被猎物折腾得遍体鳞伤。

主人公称他的真诚是“唯一的”，可以从两方面来解释，一方面是，他假定自己是丛林里中意他的猎物的唯一的猎人；另一方面，他不具备“破解黑盒的黑客思维”或是令猎物一见钟情的资本，此时他的真诚才是“唯一”的。正如我常常说的，真正的勇士除了勇气以外一无所有，那才是真正唯一的勇气。目前为止，这位猎人具备同样的勇气。

主人公随后发现了其他猎人的存在，他们越过丛林里怪物设下的陷阱显得游刃有余，与先前主人公形成了对比。说明他们具备这一“破解黑盒的黑客思维”。随后，这些人与主人公拔剑宣战。

在副歌部分，描述到他们的降临带着天选的自信，此处的天选，是指令猎物一见钟情的资本，即校园爱情优势的几大纯粹构成要素——身高、颜值、情商等等。在主人公的理解下，这一优势与前面的思维相配合，将产生绝杀的效果。

接下来，在黑暗中，几名猎手“孤独”地对弈，为什么用“孤独”这一词来描述呢，因为他们的处境非常对立，完全不存在利益共享的可能性，也就是说，他们无法展开任何形式的合作，即处于一个零和博弈的立场。后文显式地写到“殊死相拼 为这零和的游戏”再次强调了矛盾的尖锐与不可调和。

在主人公发现这几位天选的猎手之前，也因为自己思维和资本的匮乏考虑过放弃，正是一腔孤勇支撑着他克服内耗走到现在。那时的归途是他内心摇摆不定的选择，因为他明白选择了归途也许会失望、会后悔。然而此刻，归途却变得清晰了起来，这是为什么呢？因为主人公面对眼前可以藐视他的绝对强大的力量，产生了对自己深深的怀疑，已经料定自己惨败的结局。在这失败到来之前，选择可能伴随着失望和后悔的归途相比之下也更能接受，因此这归途不再摇摆不定而是变得愈发清晰起来。

随后的这一句可乃本作品点睛之笔，特别是最后的韵脚神祇(音同奇)简直让人拍案叫绝。

我先是再次引入了数理的概念来加强认同感，主人公在函数的定义域外游离，即他作为一个自变量，不在一个函数的定义域内，这是什么意思呢？其实这是把驯服猎物，即追求者追求对象的过程抽象成了一个函数表达式，求解结果的过程就是追求者追求对象的过程。显然我们知道的是，自变量不在函数的定义域内，就没有映射的结果，换句话说，追求者追求的过程是没有结果的，因为他根本就没有资格入围。顺便对应到数学上，可以设想你在求解一个非常复杂的问题，结果发现到了最后一步发现定义域的范围找错了，导致你的过程全盘否定掉，我有幸经历过这一过程。

最后一句再次将视角拉回丛林中，“他们”指的是天选的猎手们，主人公在此处把他们想象是“可敬的神祇”，是在说，具备天选优势的他们，之于主人公就像具备神威一样（这个我之前也多次在我的说说里提到过，可以想象成神威之剑、核威慑、同龄人“天生自带的Buff光环”等等），这让主人公对他们产生了敬畏。

本作品构思精妙，双线展开，叙事简单但不单调，但是对主人公迷途知返的心理活动描绘的不够有力度，我个人认为这主要和薛之谦的这首歌后半段收尾仓促有关。
